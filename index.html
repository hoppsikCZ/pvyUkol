Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • multiplatformní, objektově orientovaný programovací jazyk
  • vznikl v roce 1955, autorem je Brendan Eich
  • syntax podobný C/C++/Java, ale principiálně se od nich liší
  • Jeho největším využitím jsou dynamické a interaktivní webové stránky
  • Další využití jsou servrové aplikace a také jako nahrážka dávkových souborů přímo v OS

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. proměnné většinou definujeme jako let, nebo var
  2. pro texty používáme '', pokud specificky nepotřebujeme použít "", nebo ``
  3. středníky (;) nejsou povinné, ale měli by jsem jo používat pro ukončení příkazu
  4. pro posloupnost více příkazů využijeme bloku, označeného složenými závorkami {}
  5. do kódu lze přidávat komentáře pomocí // (jednořádkové), nebo /* [komentář] */ (víceřádkové)

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>

// 2. Vložení přímo do značky, které chceme přidat funkcionalitu
<button onclick="alert('Hello')">Click me</button>

// 3. Připojit externí soubor s JavaScriptem
<script src="script.js"></script>

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // ...
let x = 7; // celé číslo (typ number)
let x = 3.14; // reálné číslo (typ number)
let x = 8.58e-4;// zápis desetinného čísla pomocí exponentu
let x = 'Ahoj'; // typ string
let prestupny = false; // boolean (hodnota true, nebo false)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (jakýkoliv znak před = znamená: [původní proměnná] = [původní proměnná] [+, -, ...] [hodnota za =])

Logické

! (NOT), || (OR), && (AND)

Porovnávací

==, <, >, =<, > === (pokud záleží na typu proměnné)

Bitové

&, |, ^ (XOR, nastaví bit na 1, pokud je právě jeden bit 1), ~ (invertuje bity), << (posouvá bity doleva přidáním nuly z prava) >> (posouvá bity doprava přidáváním kopií prvního bitu) >>> (<< opačným směrem)

Typové

typeof (typ proměnné), instanceof (vrací true, když je objekt vlevo typ specifikovaný vpravo)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Na obrazovce vyskočí okno s textem v '' */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadej heslo:');

confirm()

/* Uživatel rozhodne, jestli chce potvrdit akci kliknutím na ok, nebo zrušit */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Popup okna by jsme moc neměli využívat, jelikož je můžou uživatelé blokovat, jsou otravné a jsou spojovány s viry a pokusy o krádež.
